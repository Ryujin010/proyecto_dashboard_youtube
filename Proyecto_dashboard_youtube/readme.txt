Proyecto: Dashboard de Tendencias de Videos en YouTube

Archivos incluidos:

- trending_by_time.csv: Datos base del dashboard
- dashboard_link.txt: Enlace al dashboard en Tableau Public
- presentacion.pdf: Informe con análisis y visualizaciones
- readme.txt: Este archivo

Instrucciones:

1. Abrir el archivo dashboard_link.txt y acceder al dashboard en Tableau Public.
2. Usar los filtros de fecha y país para explorar las tendencias.
3. Consultar presentacion.pdf para ver el análisis y respuestas clave.
4. Los datos se actualizan cada 24 horas a la medianoche UTC.
